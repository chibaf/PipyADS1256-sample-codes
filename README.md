## RasPi vs ADS1256: pin còrrespóndence between RasPi and ADS1256

GPIO#:	Description

===================

22:	Chip Select (CS)

17:	Data Ready (DRDY)

18:	RESET (GND)

27:	Power Down (PDWN)

12:	MOSI (called DIN on the slave)

13:	MISO (called DOUT on the slave)

14:	SCLK (SCK)
